# Roadmap Setelah MVP

1. Upload foto produk dan bukti pembelian.
2. Bukti pembayaran + status pembayaran.
3. Export laporan PDF/Excel.
4. Filter order per periode, per staff, per status.
5. Pencarian lanjutan customer/produk/order.
6. Notifikasi perubahan status order.
7. Sinkronisasi cloud + token auth JWT.
8. Panel admin web untuk manager.
9. Dukungan multi cabang.
10. Audit trail lengkap per perubahan data.
